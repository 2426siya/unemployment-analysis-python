# Unemployment Analysis with Python

## 📌 Objective
Analyze unemployment rate data to identify trends, seasonal patterns, and the impact of Covid-19 using Python.

## 📊 Dataset
Unemployment rate dataset containing percentage of unemployed people over time.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Analysis Performed
- Data cleaning and preprocessing
- Trend visualization
- Covid-19 impact comparison
- Monthly seasonal pattern detection
- Statistical summary

## 📈 Key Findings
- Sharp unemployment spike during Covid period
- Gradual recovery post-2021
- Seasonal variation observed across months
- Certain periods show repeated peaks

## 🏛 Policy Insights
- Need emergency employment support during crises
- Seasonal job programs can reduce recurring spikes
- Skill development programs needed in high-risk periods
